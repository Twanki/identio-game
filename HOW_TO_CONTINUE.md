# 🚀 다른 컴퓨터에서 Antigravity와 작업 이어나가는 방법

이 문서는 다른 컴퓨터나 새로운 환경에서 이 프로젝트(`identio-game`)를 다운로드하여 Antigravity(AI 코딩 어시스턴트)와 페어 프로그래밍을 계속 진행하는 가이드입니다.

---

## 1단계: GitHub에서 소스 코드 복제하기 (Clone)

새로운 컴퓨터의 터미널(Terminal)을 열고, 아래 명령어를 실행하여 깃허브에 저장된 최신 코드를 다운로드합니다.

```bash
# 1. 원하는 작업 폴더로 이동 (예: 데스크탑)
cd ~/Desktop

# 2. 저장소 복제
git clone https://github.com/Twanki/identio-game.git

# 3. 프로젝트 폴더로 이동
cd identio-game
```

---

## 2단계: Antigravity 실행 및 대화 시작

새 컴퓨터에 Antigravity 앱 또는 CLI(`agy`)가 설치되어 있다면, 복제한 `identio-game` 폴더를 활성 작업 공간(Active Workspace)으로 지정하여 새 대화를 시작합니다.

---

## 3단계: AI에게 첫 질문으로 컨텍스트 전달하기 (★ 중요)

이전 대화방의 세부 기록을 일일이 옮기지 않아도, 새 대화방에서 아래의 **"컨텍스트 복구 템플릿"**을 그대로 복사해서 저(AI)에게 첫 메시지로 보내주시면 바로 상황을 인지하고 이어서 코딩을 시작합니다.

### 📋 AI 첫 입력용 템플릿 (그대로 복사해서 붙여넣으세요)
```text
이 프로젝트는 방탈출 게임 'Identio' 웹앱 프로젝트야.
현재 깃허브와 연동하여 Vercel에 배포(https://identio-game.vercel.app)까지 완료했어.

주요 완성 기능들과 연출 스펙은 아래와 같아:
1. 시작 로딩 화면: intro_add_zero.png -> 2초 동안 로딩 게이지 충전 -> Yes 버튼 활성화(3px 아웃라인 테두리). 버튼을 터치하여 인트론 씬으로 넘어가면서 비디오 소리 즉시 자동 재생(Autoplay 보안 우회).
2. 배경 음악(BGM): game_bg.mp3가 미션 카드 오버레이 뜰 때부터 0.35 볼륨으로 루프 재생. 35초에 볼륨 페이드 아웃을 5초간 진행하여 30초 도달 시 음소거.
3. 30초 경고음: 30초 도달 시 독립 오디오 채널로 경고 알람음(time_warning.mp3) 루프 재생. 방 이동 및 효과음과 겹쳐도 소리 안 끊김.
4. 임시 음소거: 거실(lounge)의 맥북/포스터 단서 클릭 시 나오는 Oh.mp3(5초) 재생 동안 BGM 임시 음소거, 효과음 끝나거나 팝업 닫으면 음소거 자동 해제.
5. 힌트 글로우: 각 방별 2초 점등 ➡️ 소등 후 3초 대기 방식의 순차적(sequential) 힌트 구현.
   - 회의실: 책 -> 의자 -> 화분 -> 액자(Frame, x=55.5, y=30.9, w=20.0, h=6.4)
   - 거실: 맥북 -> 미니의자 -> 포스터
   - 책상: 금괴 -> 서류 -> 전선
   - 책상 모니터: 독자적인 흰색 3px 아웃라인 깜빡임 애니메이션 적용.
6. 모니터 주소창 최적화: height: 100dvh 단위를 사용하여 폰 화면에서 상하단 바에 가려 잘려 보이던 현상 해결.

위 구성 상태를 파악한 상태에서, 다음 수정 작업을 지시할 테니 확인해줘.
```
